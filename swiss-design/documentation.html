1. Overview

Swiss Design Portfolio is a single-page portfolio template built for designers, developers, and creatives. Inspired by the Swiss/International Typographic Style, it uses bold typography as the main visual element, a strict monochrome palette, generous whitespace, and grid-based layouts.

Key features: project filter by category, custom animated cursor (desktop), scroll-reveal animations, contact form with live validation, and full responsive design.

5. Adding / Editing Projects

Open index.html and find the <!-- Grid --> section. Copy and paste this card template:

<article class="work__card animate-on-scroll" data-category="CATEGORY">
  <div class="work__card-img">
    <img src="assets/img/your-image.jpg"
         alt="Short description of the project"
         loading="lazy" width="600" height="400">
  </div>
  <div class="work__card-info">
    <h3 class="work__card-title">Project Name</h3>
    <span class="work__card-tag">Category</span>
  </div>
</article>

Replace the placeholders:

  • CATEGORY — filter key (e.g. design, web, branding)
  • src — path to the project image (recommended: 600×400, place in assets/img/)
  • alt — descriptive text for accessibility and SEO
  • Project Name — display title
  • Category — visible tag label
No JavaScript changes needed — the filter reads data-* attributes dynamically via event delegation.

6. Adding Categories

In the .work__filters section of index.html, add a new button:

<button class="work__filter" data-filter="newcategory">New Category</button>

Use the same data-filter value as the data-category on your project cards.

7. Custom Cursor

The template features a custom animated cursor with two elements:

  • Outer circle — 40px, follows mouse with eased lag (0.12 lerp), uses mix-blend-mode: difference
  • Inner dot — 6px, near-instant follow (0.6 lerp)

States: hover (expands to 60px over interactive elements), click (shrinks to 32px), hidden (when mouse leaves window).

Automatically disabled on touch devices via @media (pointer: coarse).

To disable the custom cursor entirely, remove cursor.css and cursor.js references from index.html, and delete the two <div> elements with class cursor and cursor-dot.

8. Changing Colors & Typography

The template uses a strict monochrome palette. Key values in assets/css/style.css:

body {
    background-color: #0a0a0a;  /* Dark background */
    color: #f5f5f5;             /* Light text */
}

.work__card {
    background-color: #141414;  /* Card background */
}

To switch to a light theme, swap the background and text colors. For a colored accent, add it to buttons, links, and the hero__title--outline stroke.

Typography

The template uses Inter (weights 400, 700, 900) via Google Fonts. To change the font, update the <link> tag in index.html and the font-family in style.css.

9. Social Links

In the <footer> section of index.html, replace each href="#" with your actual profile URLs:

11. Performance

  • Zero external JS dependencies — only vanilla JavaScript
  • All project images use loading="lazy" with explicit width/height
  • CSS split into 3 logical files (style, cursor, animations) for maintainability
  • Intersection Observer for scroll animations (no scroll event polling)
  • prefers-reduced-motion fully respected — animations disabled for users who prefer it
  • Custom cursor uses requestAnimationFrame for smooth 60fps rendering
  • Font preconnect hints for faster Google Fonts loading

5. Adicionar / Editar Projetos

Abra index.html e encontre a seção <!-- Grid -->. Copie e cole este template de card:

<article class="work__card animate-on-scroll" data-category="CATEGORIA">
  <div class="work__card-img">
    <img src="assets/img/sua-imagem.jpg"
         alt="Descrição curta do projeto"
         loading="lazy" width="600" height="400">
  </div>
  <div class="work__card-info">
    <h3 class="work__card-title">Nome do Projeto</h3>
    <span class="work__card-tag">Categoria</span>
  </div>
</article>

Substitua os placeholders:

  • CATEGORIA — chave do filtro (ex: design, web, branding)
  • src — caminho da imagem (recomendado: 600×400, coloque em assets/img/)
  • alt — texto descritivo para acessibilidade e SEO
  • Nome do Projeto — título de exibição
  • Categoria — label visível da tag
Nenhuma alteração no JavaScript é necessária — o filtro lê os atributos data-* dinamicamente via event delegation.

7. Cursor Customizado

O template possui um cursor animado com dois elementos:

  • Círculo externo — 40px, segue o mouse com atraso suave (lerp 0.12), usa mix-blend-mode: difference
  • Ponto interno — 6px, seguimento quase instantâneo (lerp 0.6)

Estados: hover (expande para 60px sobre elementos interativos), click (encolhe para 32px), hidden (quando mouse sai da janela).

Desativado automaticamente em dispositivos touch via @media (pointer: coarse).

Para desativar o cursor customizado, remova as referências de cursor.css e cursor.js do index.html, e delete os dois <div> com classes cursor e cursor-dot.

8. Trocar Cores & Tipografia

O template usa uma paleta monocromática rigorosa. Valores principais em assets/css/style.css:

body {
    background-color: #0a0a0a;  /* Fundo escuro */
    color: #f5f5f5;             /* Texto claro */
}

.work__card {
    background-color: #141414;  /* Fundo dos cards */
}

Para um tema claro, inverta as cores de fundo e texto. Para uma cor de destaque, adicione-a aos botões, links e ao stroke do hero__title--outline.

Tipografia

O template usa Inter (pesos 400, 700, 900) via Google Fonts. Para trocar a fonte, atualize a tag <link> no index.html e o font-family no style.css.
