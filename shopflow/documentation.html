1. Overview & Features 1. Visão Geral & Recursos

ShopFlow is a conversion-focused e-commerce front-end template for fashion and tech stores. Built with Tailwind CSS v3 CDN and vanilla JavaScript — no build step, no dependencies.

  • Off-Canvas Cart Sidebar — Slides from the right, no page redirect. Items persist in localStorage
  • Product Quick View — Modal with product details, quantity selector, and add-to-cart
  • Product Tabs — New Arrivals / Best Sellers / On Sale with animated transitions
  • Product Cards — Image swap on hover, badges (New, -30%), quick action buttons
  • Hero Carousel — Full-width, autoplay (5s), swipe support, dots navigation
  • Bento Grid Categories — Mosaic layout with hover effects
  • Deal of the Day — Countdown timer, featured product with add-to-cart
  • Mobile Bottom Bar — 5-tab navigation (Home, Categories, Deals, Wishlist, Cart)
  • Search Bar — Visual suggestion dropdown (desktop), expandable on mobile
  • Toast Notifications — "Added to cart" feedback
  • Instagram Feed — Simulated social grid with hover overlay
  • Newsletter — Large input with CTA in footer
  • Payment Icons — Visa, Mastercard, PIX, Boleto, Amex, Elo

ShopFlow é um template front-end de e-commerce focado em conversão para lojas de moda e tecnologia. Construído com Tailwind CSS v3 CDN e JavaScript puro — sem build, sem dependências.

3. Product Data Structure 3. Estrutura de Dados dos Produtos

Products are defined in the PRODUCTS object at the top of store.js. It has three arrays:

  • new — New Arrivals (Novidades tab)
  • best — Best Sellers (Mais Vendidos tab)
  • sale — On Sale (Ofertas tab)

Each product object:

{
    id: 'n1',                    // Unique identifier
    name: 'Camiseta Oversized',  // Display name
    price: 129,                  // Current price (number)
    oldPrice: null,              // Original price before discount (null if no sale)
    badge: 'New',                // Badge text ('New', '-40%', null)
    reviews: 24,                 // Number of reviews (for Quick View)
    desc: 'Product description', // Description text (for Quick View)
    installments: '3x de R$ 43'  // Installment text
}

To add your own products, simply edit the arrays. Each tab shows 8 products (4-column grid on desktop, 2-column on mobile).

Tip: To load products from an API, replace the static PRODUCTS object with a fetch() call in the init() function.

Os produtos são definidos no objeto PRODUCTS no topo do store.js. Ele possui três arrays:

  • new — Novidades (aba Novidades)
  • best — Mais Vendidos (aba Mais Vendidos)
  • sale — Ofertas (aba Ofertas)

Cada objeto de produto:

{
    id: 'n1',                      // Identificador único
    name: 'Camiseta Oversized',    // Nome de exibição
    price: 129,                    // Preço atual (número)
    oldPrice: null,                // Preço original antes do desconto (null se sem promoção)
    badge: 'New',                  // Texto do badge ('New', '-40%', null)
    reviews: 24,                   // Número de avaliações (para Quick View)
    desc: 'Descrição do produto',  // Texto de descrição (para Quick View)
    installments: '3x de R$ 43'    // Texto de parcelamento
}

Para adicionar seus próprios produtos, basta editar os arrays. Cada aba mostra 8 produtos (grid 4 colunas no desktop, 2 no mobile).

Dica: Para carregar produtos de uma API, substitua o objeto estático PRODUCTS por uma chamada fetch() na função init().

4. Cart Sidebar (Off-Canvas) 4. Carrinho Lateral (Off-Canvas)

The cart sidebar slides from the right without redirecting to another page. Key behaviors:

  • Opens automatically when an item is added to cart
  • Persists items in localStorage under key shopflow-cart
  • Quantity controls — increase/decrease per item
  • Remove button — X icon on each item
  • Empty state — Shows bag icon and "Continue shopping" link
  • Closes via — Close button, overlay click, or ESC key
  • Prevents body scroll when open
  • Opens from: header cart icon, mobile bottom bar cart button, or after adding any item

Badge counters update on: header cart icon, mobile bottom bar cart icon.

O carrinho lateral desliza da direita sem redirecionar para outra página. Comportamentos principais:

  • Abre automaticamente ao adicionar um item ao carrinho
  • Persiste itens no localStorage na chave shopflow-cart
  • Controles de quantidade — aumentar/diminuir por item
  • Botão remover — Ícone X em cada item
  • Estado vazio — Mostra ícone de sacola e link "Continuar comprando"
  • Fecha por — Botão fechar, clique no overlay, ou tecla ESC
  • Previne scroll do body quando aberto
  • Abre de: ícone do carrinho no header, botão do carrinho na barra mobile, ou após adicionar qualquer item

Badges de contagem atualizam em: ícone do carrinho no header, ícone do carrinho na barra mobile.

5. Quick View Modal 5. Modal Quick View

The Quick View modal opens when clicking the eye icon on a product card (visible on hover). It shows:

  • Product image placeholder (left column)
  • Badge (New or sale percentage)
  • Product name and star rating
  • Price with old price crossed out (if on sale)
  • Description text
  • Quantity selector (+/-)
  • "Add to Cart" button — adds the selected quantity and opens the cart sidebar

Closes via: X button, overlay click, or ESC key.

O modal Quick View abre ao clicar no ícone de olho no card do produto (visível no hover). Mostra:

  • Placeholder da imagem do produto (coluna esquerda)
  • Badge (New ou porcentagem de desconto)
  • Nome do produto e classificação por estrelas
  • Preço com preço antigo riscado (se em oferta)
  • Texto de descrição
  • Seletor de quantidade (+/-)
  • Botão "Adicionar ao Carrinho" — adiciona a quantidade selecionada e abre o carrinho

Fecha por: botão X, clique no overlay, ou tecla ESC.

7. Color Customization 7. Personalização de Cores

Colors are in the Tailwind config inside index.html head:

  • shop-* — Primary slate/navy palette (backgrounds, text, borders)
  • accent-* — Amber/gold accent (CTAs, highlights, badges, logo)
  • sale-* — Red for sale badges and promotional elements
  • green-* — Tailwind default greens (success states)

Also update the corresponding colors in ecommerce.css (product card styles, toast background, etc.).

As cores estão na config do Tailwind dentro do index.html head:

  • shop-* — Paleta primária slate/navy (fundos, textos, bordas)
  • accent-* — Destaque âmbar/dourado (CTAs, destaques, badges, logo)
  • sale-* — Vermelho para badges de oferta e elementos promocionais
  • green-* — Verdes padrão do Tailwind (estados de sucesso)

Atualize também as cores correspondentes no ecommerce.css (estilos dos cards, fundo do toast, etc.).

9. JavaScript API 9. API JavaScript

The ShopFlow global object exposes these public methods:

O objeto global ShopFlow expõe estes métodos públicos:

ShopFlow.addToCart(id, tab)      // Add product to cart
ShopFlow.removeFromCart(id)      // Remove product from cart
ShopFlow.updateQty(id, delta)    // Change quantity (+1 or -1)
ShopFlow.openCart()              // Open cart sidebar
ShopFlow.closeCart()             // Close cart sidebar
ShopFlow.openQuickView(id, tab)  // Open quick view modal
ShopFlow.closeQuickView()        // Close quick view modal
ShopFlow.qvQty(delta)            // Change quick view quantity

Cart data is stored in localStorage as JSON array under key shopflow-cart.

Os dados do carrinho são armazenados no localStorage como array JSON na chave shopflow-cart.
