1. Overview

Chronicle is a premium blog/magazine template designed for tech content. Inspired by Medium and Vercel Blog, it focuses on readability, performance, and reader retention.

Key features include: real dark mode, reading progress bar, floating share buttons, beautiful code blocks with copy functionality, responsive design, and SEO-optimized semantic HTML.

4. Pages

index.html — Home

  • Header: Sticky navbar with logo, nav links, expandable search bar (Ctrl+K), dark mode toggle, mobile off-canvas menu, and horizontal scrolling category tags.
  • Featured Post (Hero): Large 3/5 + 2/5 grid layout with image, category badge, title, summary, author info and reading time.
  • Article Grid: Asymmetric grid with large (2-col) and small (1-col) cards. Each card has: image, category badge, title, excerpt, author avatar, and reading time.
  • Sidebar (Sticky): Newsletter signup box, Trending Topics (numbered list), and Ad Space placeholder (300×250).

post.html — Single Article

  • Reading Progress Bar: Fixed at the very top, gradient from accent to purple.
  • Floating Share Buttons: Fixed on the left side (desktop). Twitter/X, LinkedIn, Copy Link, and Back to Top.
  • Article Content: Custom prose typography with serif body, heading hierarchy, blockquotes, inline code, code blocks with syntax highlighting and copy button, info/warning callouts.
  • Author Bio: Card with avatar, name, social link, bio and follow/profile buttons.
  • Read Next: 3-column grid of related articles at the bottom.

category.html — Category Listing

  • Category header with icon, name, description, article count, and sort dropdown.
  • List-style article cards with 2/5 image + 3/5 content layout.
  • Pagination with numbered buttons and prev/next arrows.
  • Sidebar with all categories (with counts), newsletter, and ad space.

5. Dark Mode

Dark mode is implemented using Tailwind's class strategy. The theme preference is persisted in localStorage and respects the OS prefers-color-scheme on first visit. 

// Theme is stored as:
localStorage.setItem('chronicle-theme', 'dark' | 'light');

// Colors:
// Light: bg-surface-light (#ffffff), text-ink-800 (#212529)
// Dark:  bg-surface-dark (#0f1117), text-ink-200 (#e9ecef)
// Alt:   bg-surface-dark-alt (#161922) — cards, sidebar

6. Customization

Changing the Accent Color

Edit the Tailwind config in the <script> tag of each HTML file: 

// In tailwind.config:
colors: {
    accent: {
        DEFAULT: '#6366f1', // Change this to your brand color
        light: '#818cf8',   // Lighter variant
        dark: '#4f46e5',    // Darker variant (hover)
    }
}

Also update style.css where the hex #6366f1 appears (focus outlines, selection color, code blocks).

Changing Fonts

Update the Google Fonts link and Tailwind config: 

// Tailwind config:
fontFamily: {
    sans: ['Inter', ...],          // Titles & UI
    serif: ['Merriweather', ...],  // Body text
    mono: ['JetBrains Mono', ...], // Code
}

// Also update style.css where font-family is specified

Images

All images use Unsplash URLs for demo purposes. Replace them with your actual images. Recommended sizes:

  • Featured/Hero: 900×600px
  • Article post cover: 1200×600px
  • Card thumbnails (large): 600×400px
  • Card thumbnails (small): 400×260px

7. Code Blocks

Code blocks are styled with a dark theme (matching the dark mode background) for consistency. Each block includes:

  • Decorative traffic light dots (macOS style)
  • Language label (e.g., "javascript")
  • Copy button with feedback ("Copiado!")

To add real syntax highlighting, integrate a library like Prism.js or Highlight.js. The dark theme will pair naturally with the existing code block styles. 

<!-- Add Prism.js for syntax highlighting -->
<link href="https://cdn.jsdelivr.net/npm/prismjs@1/themes/prism-tomorrow.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/prismjs@1/prism.min.js"></script>
<script src="https://cdn.jsdelivr.net/npm/prismjs@1/components/prism-javascript.min.js"></script>

8. SEO & Semantics

Chronicle uses semantic HTML5 tags throughout:

  • <article> — Each post card and the main article content
  • <aside> — Sidebar sections (newsletter, trending, ads)
  • <time datetime> — All publication dates with ISO format
  • <nav aria-label> — Navigation with accessible labels
  • <figure> / <figcaption> — Post cover images with captions
  • <header> / <main> / <footer> — Page landmarks

Open Graph meta tags are included for social sharing. Update the og:image URL with your actual image.

9. Performance

  • Zero dependencies — No jQuery, no heavy libraries. Only Tailwind CDN + vanilla JS.
  • Lazy loading — All non-hero images use loading="lazy".
  • Passive listeners — Scroll events use { passive: true }.
  • Font preconnect — Google Fonts domains are preconnected.
  • Staggered reveal — Cards use Intersection Observer for staggered fade-in.
  • Print styles — Header, footer, share buttons and progress bar are hidden in print.

For production, replace the Tailwind CDN with a build step (npx tailwindcss -o style.min.css --minify) for significantly smaller CSS.
