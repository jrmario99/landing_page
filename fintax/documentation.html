1. File Structure

fintax/
├── index.html              ← Main page
├── documentation.html      ← This file
└── assets/
    ├── css/style.css       ← All styles (CSS variables)
    ├── js/main.js          ← Calculator, timeline, reveal
    └── img/
        ├── favicon.svg
        └── (your images)

2. Colors & Branding

Open assets/css/style.css and edit the :root variables at the top:

/* Primary Blue */
--blue-600: #2563eb;

/* Green (money/success) */
--green-600: #16a34a;

All components using --blue-* and --green-* will update accordingly.

3. Navbar & Client Login

  • The "Área do Cliente" button is in navbar__actions — replace href="#" with your actual login URL
  • The navbar is fixed with blur backdrop and gains shadow on scroll
  • Mobile menu slides open below the navbar

4. Hero Section

  • Replace the image placeholder with a photo of a professional (recommended: 600×700px)
  • The mini bar charts are pure CSS — edit the --h values to change bar heights
  • The floating "38% savings" card animates with CSS — edit text in hero__float-card
  • The green badge ("Vagas limitadas") has a pulsing dot animation

5. Services Grid

Each service uses a different color icon. To change an icon's SVG, replace the content inside service-box__icon. Color classes available:

service-box__icon--blue
service-box__icon--green
service-box__icon--purple
service-box__icon--amber
service-box__icon--rose
service-box__icon--cyan

6. Tax Calculator (Hook)

This is the key conversion element. The calculator uses simplified tax rates for demonstration.

To customize the tax logic:

  1. Open assets/js/main.js
  2. Find the comment // EDITABLE: Tax calculation logic
  3. Replace the rate values with your actual formulas
// Current simplified rates:
simples:   12%
presumido: 16%
real:      25%
naosei:    15%

// Optimized = current × 0.65 (~35% savings)
// Replace these with your real formulas

The result includes animated comparison bars and a CTA to the pricing section.

7. Process Timeline

The timeline has a "draw-on-scroll" effect: a gradient line fills as the user scrolls, and markers turn blue when reached. This is handled automatically by the Intersection Observer in main.js.

To add a step, duplicate a timeline__item block and update the number and content.

8. Pricing Tables

  • 3 cards: Starter, Professional (featured), Enterprise
  • The middle card has a pricing-card--featured class for the blue highlight
  • Each feature uses class="included" or class="excluded" for checkmark/X styling
  • Edit prices, features, and CTA links directly in the HTML

9. Blog / News

3 article cards with image placeholder, category badge, date, title, and excerpt. Replace with real content and link each card's <a href> to actual blog posts.

10. Footer & Security Badges

  • 4 columns: Brand, Quick Links, Services, Contact
  • Security badges (SSL, LGPD, CRC) are in footer__badges — add or remove as needed
  • Update all links and contact information in the HTML

2. Cores e Marca

Abra assets/css/style.css e edite as variáveis :root no topo:

/* Azul principal */
--blue-600: #2563eb;

/* Verde (dinheiro/sucesso) */
--green-600: #16a34a;

3. Navbar e Login do Cliente

  • O botão "Área do Cliente" está em navbar__actions — substitua href="#" pela URL real do login
  • A navbar é fixa com backdrop blur e ganha sombra ao rolar
  • O menu mobile desliza abaixo da navbar

4. Seção Hero

  • Substitua o placeholder pela foto de um profissional (recomendado: 600×700px)
  • Os mini gráficos de barra são CSS puro — edite os valores --h para alterar as alturas
  • O card flutuante "38% economia" anima com CSS — edite o texto em hero__float-card
  • O badge verde ("Vagas limitadas") tem uma animação de pulso

5. Grid de Serviços

Cada serviço usa um ícone de cor diferente. Para trocar um ícone SVG, substitua o conteúdo dentro de service-box__icon. Classes de cor disponíveis:

service-box__icon--blue
service-box__icon--green
service-box__icon--purple
service-box__icon--amber
service-box__icon--rose
service-box__icon--cyan

6. Calculadora de Impostos (Hook)

Este é o elemento-chave de conversão. A calculadora usa alíquotas simplificadas para demonstração.

Para personalizar a lógica fiscal:

  1. Abra assets/js/main.js
  2. Encontre o comentário // EDITABLE: Tax calculation logic
  3. Substitua os valores de alíquota pelas suas fórmulas reais
// Alíquotas simplificadas atuais:
simples:   12%
presumido: 16%
real:      25%
naosei:    15%

// Otimizado = atual × 0.65 (~35% economia)
// Substitua por suas fórmulas reais

O resultado inclui barras de comparação animadas e um CTA para a seção de preços.

7. Timeline de Processo

A timeline tem efeito de "desenho ao rolar": uma linha gradiente preenche conforme o usuário rola, e os marcadores ficam azuis quando atingidos. Tudo é gerenciado automaticamente pelo Intersection Observer no main.js.

Para adicionar um passo, duplique um bloco timeline__item e atualize o número e o conteúdo.

8. Tabelas de Preços

  • 3 cards: Starter, Professional (destaque), Enterprise
  • O card do meio tem a classe pricing-card--featured para o destaque azul
  • Cada feature usa class="included" ou class="excluded" para estilo de check/X
  • Edite preços, funcionalidades e links CTA diretamente no HTML

9. Blog / Notícias

3 cards de artigo com placeholder de imagem, badge de categoria, data, título e resumo. Substitua pelo conteúdo real e vincule cada <a href> ao post real do blog.

10. Footer e Selos de Segurança

  • 4 colunas: Marca, Links Rápidos, Serviços, Contato
  • Selos de segurança (SSL, LGPD, CRC) estão em footer__badges — adicione ou remova conforme necessário
  • Atualize todos os links e informações de contato no HTML
