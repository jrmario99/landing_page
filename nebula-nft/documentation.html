1. Overview

NebulaNFT is a single-page landing page for NFT collections, Web3 projects, and blockchain communities. Features a full-screen animated starfield canvas, glassmorphism UI, animated roadmap timeline, team grid, FAQ accordion, and a wallet connect modal.

Built with Tailwind CSS and vanilla JavaScript. The starfield is rendered on a <canvas> element for smooth 60fps animation.

5. Customizing the Roadmap

Each phase is a .roadmap-item div with data-phase attribute. Edit the label, title, and milestone <li> items.

Adding a New Phase

  1. Copy an existing .roadmap-item block
  2. Update data-phase to the next number
  3. Choose side: odd phases = left, even phases = right
  4. Pick a dot color: bg-holo-cyan, bg-holo-purple, bg-holo-pink, or bg-holo-blue
The connecting line and scroll animations adjust automatically — no JS changes needed.

6. Customizing the Team

Team members are in <section id="team">. Each member is a .glass-card. Edit the avatar letter, name, role, gradient colors (from-holo-* / to-holo-*), and social link URLs.

To add a member, copy an existing card and adjust the grid columns (e.g., md:grid-cols-5).

7. Customizing the FAQ

Each FAQ entry is a .faq-item div. Edit the question in .faq-toggle span and the answer in .faq-content p. To add entries, copy a block — accordion behavior is applied automatically.

8. Changing Colors

Colors are defined in two places:

Tailwind Config (index.html)

colors: {
  nebula: { 900: '#0a0a1a', 800: '#111128', ... },
  holo: {
    cyan: '#00f0ff',    // Primary accent
    purple: '#a855f7',  // Secondary
    pink: '#ec4899',    // Tertiary
    blue: '#3b82f6',    // Quaternary
  }
}

CSS Gradients (style.css)

Update gradient values in .holo-text, .btn-holo, and .timeline-line classes to match.

5. Personalizar o Roadmap

Cada fase é um .roadmap-item com atributo data-phase. Edite o label, título e itens <li>.

Adicionar Nova Fase

  1. Copie um bloco .roadmap-item existente
  2. Atualize data-phase para o próximo número
  3. Escolha o lado: fases ímpares = esquerda, pares = direita
  4. Escolha cor do dot: bg-holo-cyan, bg-holo-purple, bg-holo-pink ou bg-holo-blue
A linha conectora e animações de scroll se ajustam automaticamente — sem mudanças no JS.

6. Personalizar a Equipe

Os membros estão em <section id="team">. Cada membro é um .glass-card. Edite a letra do avatar, nome, cargo, cores do gradiente (from-holo-* / to-holo-*) e URLs dos links sociais.

Para adicionar um membro, copie um card existente e ajuste as colunas do grid (ex: md:grid-cols-5).

7. Personalizar o FAQ

Cada entrada é um .faq-item. Edite a pergunta em .faq-toggle span e a resposta em .faq-content p. Para adicionar, copie um bloco — o comportamento do accordion é aplicado automaticamente.
