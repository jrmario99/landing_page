1. Overview1. Visao Geral

NexusAdmin is a professional admin dashboard template designed for SaaS backends, CRMs, and ERPs. Built with a clean, modular, Stripe-inspired design philosophy emphasizing high contrast and readability.

Key features include: responsive sidebar navigation, real-time stat cards, interactive ApexCharts (revenue & traffic), a transactions table, a full settings page with tabbed forms, and native dark mode with smooth transitions.

NexusAdmin e um template profissional de dashboard admin projetado para backends SaaS, CRMs e ERPs. Construido com um design limpo, modular e inspirado no Stripe, com enfase em alto contraste e legibilidade.

Recursos principais: navegacao sidebar responsiva, cards de estatisticas em tempo real, graficos interativos ApexCharts (receita e trafego), tabela de transacoes, pagina de configuracoes completa com abas, e modo escuro nativo com transicoes suaves.

4. Layout Architecture4. Arquitetura de Layout

The dashboard uses a fixed sidebar + scrollable main content layout:

  • Sidebar — Fixed left, w-64 (256px). Contains logo, nav links, and "Upgrade to Pro" CTA.
  • Main Area — lg:ml-64 offset. Contains the sticky top navbar and scrollable content.
  • Top Navbar — Sticky with search, dark mode toggle, notifications, and profile dropdown.

On mobile (< lg), the sidebar is hidden and toggled via a hamburger menu with an overlay.

O dashboard usa um layout de sidebar fixa + conteudo principal rolavel:

  • Sidebar — Fixa a esquerda, w-64 (256px). Contem logo, links de navegacao e CTA "Upgrade to Pro".
  • Area Principal — Offset lg:ml-64. Contem a navbar sticky e conteudo rolavel.
  • Navbar Superior — Sticky com busca, toggle modo escuro, notificacoes e dropdown de perfil.

No mobile (< lg), a sidebar fica oculta e e ativada pelo menu hamburger com overlay.

5. Sidebar Customization5. Personalizacao da Sidebar

Navigation links are inside the <nav> element in the sidebar. Each link uses the .nav-link class. To add a new link:

<a href="new-page.html" class="nav-link">
    <i data-lucide="icon-name" class="w-5 h-5"></i>
    <span>Page Name</span>
</a>

Mark the active page by adding the active class to its link. Browse available icons at lucide.dev/icons.

Os links de navegacao ficam dentro do elemento <nav> na sidebar. Cada link usa a classe .nav-link. Para adicionar um novo link:

<a href="nova-pagina.html" class="nav-link">
    <i data-lucide="nome-icone" class="w-5 h-5"></i>
    <span>Nome da Pagina</span>
</a>

Marque a pagina ativa adicionando a classe active ao link. Consulte icones disponiveis em lucide.dev/icons.

6. Charts Configuration6. Configuracao de Graficos

Charts are powered by ApexCharts and initialized in assets/js/main.js. Two charts are included:

  • Revenue Overview — Area chart with Revenue vs Expenses series (12 months)
  • Traffic Sources — Donut chart with Direct, Organic, Referral, Social segments

To modify chart data, edit the series arrays in main.js. The charts automatically adapt to dark mode via a MutationObserver that watches for class changes on the <html> element.

Changing Chart Colors

// In main.js, update the colors array:
colors: ['#6366f1', '#06b6d4'],  // Revenue chart
colors: ['#6366f1', '#06b6d4', '#f59e0b', '#10b981'],  // Traffic chart

Os graficos sao alimentados pelo ApexCharts e inicializados em assets/js/main.js. Dois graficos estao inclusos:

  • Visao de Receita — Grafico area com series Receita vs Despesas (12 meses)
  • Fontes de Trafego — Grafico donut com segmentos Direto, Organico, Referencia, Social

Para modificar os dados, edite os arrays series em main.js. Os graficos se adaptam automaticamente ao modo escuro via MutationObserver que monitora mudancas de classe no elemento <html>.

Alterando Cores dos Graficos

// Em main.js, atualize o array colors:
colors: ['#6366f1', '#06b6d4'],  // Grafico de receita
colors: ['#6366f1', '#06b6d4', '#f59e0b', '#10b981'],  // Grafico de trafego

7. Dark Mode7. Modo Escuro

Dark mode is managed by Alpine.js using Tailwind's darkMode: 'class' strategy:

  1. On load, a FOUC-prevention script in <head> reads localStorage('nexus-dark') and sets the dark class immediately.
  2. Alpine.js $watch syncs the toggle state to localStorage and toggles the class.
  3. ApexCharts detect the class change via MutationObserver and re-theme automatically.

Light mode: white cards (bg-white) on bg-slate-50. Dark mode: bg-slate-800 cards on bg-slate-900.

O modo escuro e gerenciado pelo Alpine.js usando a estrategia darkMode: 'class' do Tailwind:

  1. No carregamento, um script anti-FOUC no <head> le localStorage('nexus-dark') e define a classe dark imediatamente.
  2. O $watch do Alpine.js sincroniza o estado do toggle com localStorage e alterna a classe.
  3. O ApexCharts detecta a mudanca via MutationObserver e re-aplica o tema automaticamente.

Modo claro: cards brancos (bg-white) sobre bg-slate-50. Modo escuro: cards bg-slate-800 sobre bg-slate-900.

8. Forms & Settings8. Formularios & Configuracoes

The settings page (settings.html) features an Alpine.js tab system with 4 panels:

  • Profile — Avatar upload, name, email, phone, bio fields
  • Security — Password change form + 2FA toggle switch
  • Billing — Current plan, payment method, billing history table
  • Notifications — 5 notification preferences with toggle switches

Toggle switches use the .toggle-switch CSS class. Toggle the active class to change state.

A pagina de configuracoes (settings.html) possui um sistema de abas Alpine.js com 4 paineis:

  • Perfil — Upload de avatar, nome, email, telefone, bio
  • Seguranca — Formulario de senha + toggle 2FA
  • Cobranca — Plano atual, metodo de pagamento, historico de cobranca
  • Notificacoes — 5 preferencias com toggle switches

Os toggle switches usam a classe CSS .toggle-switch. Alterne a classe active para mudar o estado.

9. Color Customization9. Personalizacao de Cores

The primary brand color is defined as nexus in the Tailwind config inside each HTML file:

nexus: {
    50: '#eef2ff', 100: '#e0e7ff', 200: '#c7d2fe',
    300: '#a5b4fc', 400: '#818cf8', 500: '#6366f1',
    600: '#4f46e5', 700: '#4338ca', 800: '#3730a3',
    900: '#312e81', 950: '#1e1b4b',
}

Replace these values to match your brand. Also update the gradient colors used in the sidebar logo and "Upgrade to Pro" card (from-nexus-500 to-cyan-500).

A cor primaria da marca e definida como nexus na config do Tailwind dentro de cada arquivo HTML:

nexus: {
    50: '#eef2ff', 100: '#e0e7ff', 200: '#c7d2fe',
    300: '#a5b4fc', 400: '#818cf8', 500: '#6366f1',
    600: '#4f46e5', 700: '#4338ca', 800: '#3730a3',
    900: '#312e81', 950: '#1e1b4b',
}

Substitua esses valores para combinar com sua marca. Atualize tambem as cores de gradiente usadas no logo da sidebar e no card "Upgrade to Pro" (from-nexus-500 to-cyan-500).
