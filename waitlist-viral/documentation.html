1. Overview

Waitlist Viral is a single-page coming soon / pre-launch landing page designed to capture emails and encourage viral sharing. It features an animated particle background, progress bar with subscriber count, email form with validation, and social share buttons (WhatsApp, Twitter/X, Telegram, Copy Link).

5. Configuration (script.js)

Open assets/js/script.js and edit the CONFIG object at the top:

const CONFIG = {
  currentSubscribers: 847,     // Current count shown
  goalSubscribers: 1000,       // Goal number
  siteUrl: 'https://seusite.com',
  shareText: 'Algo incrível está chegando!...',
  simulateSubmit: true,        // Set false for real backend
};
Set simulateSubmit: false when you connect a real email service (Mailchimp, ConvertKit, etc.).

6. Email Service Integration

The template supports integration with:

  • Mailchimp — JSONP integration via embedded form URL
  • ConvertKit — Fetch API to ConvertKit forms endpoint
  • Custom API — POST request to any endpoint

Replace the submitEmail() function in script.js with the integration code for your chosen service. The function receives the validated email and should call showSuccess(email) on success or showError(msg) on failure.

7. Editing Text & Copy

All text is in index.html. Key elements:

  • Badge — "Pré-lançamento" text in .badge
  • Headline — <h1 class="hero-title"> with gradient text
  • Subtitle — <p class="subtitle">
  • Progress label — "Comunidade crescendo" in .progress-label
  • CTA button — "Quero participar" in .btn-text
  • Success message — In #successMessage

8. Changing Colors

Edit CSS variables in assets/css/style.css at :root:

:root {
  --bg: #0a0a0b;        /* Page background */
  --surface: #141416;   /* Card/form background */
  --border: #222228;    /* Border color */
  --accent: #a855f7;    /* Primary purple accent */
  --accent-glow: ...;   /* Glow effects */
}

9. Social Sharing

Share buttons appear after successful email signup. The share text and URL are configured in the CONFIG object in script.js. Links are generated dynamically for WhatsApp, Twitter/X, and Telegram.
