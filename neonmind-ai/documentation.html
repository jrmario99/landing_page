1. Overview 1. Visão Geral

NeonMind AI is a modern, fully static landing page template designed for AI-powered SaaS products. It features a striking dark neon aesthetic with purple, cyan and pink accent colors, delivering a cutting-edge visual identity that resonates with tech audiences. NeonMind AI é um template de landing page moderno, totalmente estático, projetado para produtos SaaS baseados em inteligência artificial. Apresenta uma estética dark neon marcante com cores de destaque em roxo, ciano e rosa, entregando uma identidade visual de ponta para audiências tech.

The template is built without any JavaScript frameworks — only vanilla JS is used for interactive components. It requires no build step, no package manager and no server: just open index.html in a browser or deploy to any static host. O template é construído sem frameworks JavaScript — apenas JS vanilla é utilizado para componentes interativos. Não requer etapa de build, gerenciador de pacotes ou servidor: basta abrir index.html no navegador ou fazer deploy em qualquer host estático.

5. Typewriter Effect 5. Efeito Typewriter

The hero headline cycles through a list of AI-related words using a pure-JS typewriter that types each word character by character, pauses, then deletes it before moving to the next word. A blinking cursor is appended with CSS. O título do hero percorre uma lista de palavras relacionadas à IA usando um typewriter em JS puro que digita cada palavra caractere a caractere, pausa e então a apaga antes de passar para a próxima. Um cursor piscante é adicionado com CSS. 

// script.js — Typewriter logic// script.js — Lógica do Typewriter
const words   = ['Intelligence', 'Creativity', 'Productivity', 'Innovation'];
const target  = document.getElementById('typewriter');
let wordIndex = 0;
let charIndex = 0;
let isDeleting = false;

function type() {
  const current = words[wordIndex];
  const speed   = isDeleting ? 60 : 120;

  if (!isDeleting) {
    target.textContent = current.slice(0, ++charIndex);
    if (charIndex === current.length) {
      isDeleting = true;
      return setTimeout(type, 1800); // pause after full word// pausa após palavra completa
    }
  } else {
    target.textContent = current.slice(0, --charIndex);
    if (charIndex === 0) {
      isDeleting = false;
      wordIndex  = (wordIndex + 1) % words.length;
    }
  }
  setTimeout(type, speed);
}
type();
Timing Temporização
  • Typing speed:Velocidade de digitação: 120ms/char
  • Delete speed:Velocidade de deleção: 60ms/char
  • Pause (full word):Pausa (palavra completa): 1800ms
Customization Customização
  • Edit the words array to change cycling words. Edite o array words para mudar as palavras.
  • Adjust timing constants to change speed/pause. Ajuste as constantes de tempo para mudar velocidade/pausa.

6. Comparison Slider 6. Slider de Comparação

The demo section features a before/after image comparison slider. Dragging the handle reveals either the "Before AI" or "After AI" image. It supports both mouse and touch input for full mobile compatibility. A seção de demo apresenta um slider de comparação de imagens antes/depois. Arrastar o handle revela a imagem "Antes da IA" ou "Depois da IA". Suporta mouse e touch para compatibilidade total em mobile. 

// script.js — Comparison slider (simplified)// script.js — Slider de comparação (simplificado)
const slider   = document.getElementById('comparison-slider');
const handle   = document.getElementById('slider-handle');
const afterImg = document.getElementById('after-image');
let isDragging = false;

function updateSlider(clientX) {
  const rect   = slider.getBoundingClientRect();
  const offset = Math.min(Math.max(clientX - rect.left, 0), rect.width);
  const pct    = (offset / rect.width) * 100;
  afterImg.style.clipPath = `inset(0 ${100 - pct}% 0 0)`;
  handle.style.left = `${pct}%`;
}

// Mouse events// Eventos de mouse
handle.addEventListener('mousedown', () => isDragging = true);
document.addEventListener('mousemove', e => { if (isDragging) updateSlider(e.clientX); });
document.addEventListener('mouseup',   () => isDragging = false);

// Touch events// Eventos de touch
handle.addEventListener('touchstart', () => isDragging = true, { passive: true });
document.addEventListener('touchmove', e => {
  if (isDragging) updateSlider(e.touches[0].clientX);
}, { passive: true });
document.addEventListener('touchend', () => isDragging = false);
How to replace images: Como substituir as imagens: Find the before-image and after-image elements in index.html and update their src attributes with your own URLs or local paths. Both images should be the same dimensions for best results. Encontre os elementos before-image e after-image em index.html e atualize seus atributos src com suas próprias URLs ou caminhos locais. Ambas as imagens devem ter as mesmas dimensões para melhores resultados.

7. Glass Cards 7. Glass Cards

Glass cards are the primary UI surface throughout the template. They use the glassmorphism technique — a semi-transparent background with a backdrop blur — to create a frosted-glass effect that sits naturally on the dark neon background. Glass cards são a principal superfície de UI em todo o template. Utilizam a técnica de glassmorphism — fundo semi-transparente com blur de backdrop — para criar um efeito de vidro fosco que se encaixa naturalmente no fundo dark neon. 

/* style.css — Glass card definition *//* style.css — Definição do glass card */
.glass-card {
  background:         rgba(255, 255, 255, 0.04);
  border:             1px solid rgba(255, 255, 255, 0.08);
  backdrop-filter:    blur(12px);
  -webkit-backdrop-filter: blur(12px);
  border-radius:      1rem;
  transition:         border-color 0.3s ease, box-shadow 0.3s ease;
}

.glass-card:hover {
  border-color: rgba(176, 38, 255, 0.35);
  box-shadow:   0 0 28px rgba(176, 38, 255, 0.12);
}

Key properties Propriedades principais

  • backdrop-filter: blur(12px) — blur depthprofundidade do blur
  • rgba(255,255,255,0.04) — 4% white tinttinta branca a 4%
  • rgba(255,255,255,0.08) — subtle borderborda sutil
  • Hover border brightens to neon purpleBorda no hover clareia para roxo neon

Usage Uso

  • Add class="glass-card" to any container. Adicione class="glass-card" a qualquer container.

11. Performance 11. Performance

Because the template is fully static with CDN-loaded assets, it is fast out of the box. The following optimizations are already applied in index.html. Como o template é totalmente estático com assets carregados via CDN, ele é rápido por padrão. As seguintes otimizações já estão aplicadas em index.html.

DNS Preconnect Pré-conexão DNS 

<link rel="preconnect" href="https://fonts.googleapis.com" />
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />

Establishes early connections to Google Fonts servers, reducing font load latency. Estabelece conexões antecipadas com servidores do Google Fonts, reduzindo a latência de carregamento das fontes.

CDN Resources Recursos via CDN

  • Tailwind CSS CDN — served from Tailwind's global CDN with HTTP/2 push and edge caching. — servido pelo CDN global do Tailwind com HTTP/2 push e cache em edge.
  • Google Fonts — cached aggressively by browsers after first load; uses display=swap to avoid render-blocking. — cacheadas agressivamente pelo navegador após o primeiro carregamento; usa display=swap para evitar bloqueio de renderização.

Minimal Custom CSS CSS Customizado Mínimo

Everything else uses Tailwind, keeping the custom CSS footprint tiny. style.css define apenas regras que não podem ser expressas com classes utilitárias do Tailwind: animações keyframe, backdrop-filter para glassmorphism, a técnica de clipe de gradiente de texto e tema da barra de rolagem. JS vanilla para o typewriter e slider significa zero peso de bundle de framework. O script.js completo tem poucos kilobytes — muito menor do que qualquer equivalente em framework.
