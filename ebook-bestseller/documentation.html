1. Overview

E-Book Bestseller is a single-page sales landing page designed to sell e-books, digital courses, and info products. It follows proven direct-response copywriting structure: urgency bar, hero with 3D book cover, pain/problem agitation, chapter outline, social proof, bonuses, pricing with anchor, guarantee, FAQ, and final CTA.

Built with Tailwind CSS and vanilla JavaScript. Zero dependencies beyond the Tailwind CDN.

4. Page Sections

  • Urgency Bar — Fixed top bar with countdown timer and "offer ends" message
  • Hero — Headline, subheadline, 3D book cover with CSS perspective, dual CTA buttons
  • Pain/Problem — Agitation section highlighting the reader's struggles
  • What You'll Learn — Chapter-by-chapter checklist with checkmarks
  • Testimonials — 3 client review cards with star ratings and author info
  • Author Bio — "Meet the Author" section with photo placeholder and credentials
  • Bonuses — 3 bonus cards with value anchoring (e.g., "Value: $47")
  • Pricing — Price anchor (original vs. sale), buy button, feature list, and countdown

  • FAQ — Accordion-style questions with expand/collapse
  • Final CTA — Last call to action section with urgency
  • Footer — Copyright, privacy, terms, contact links

5. Editing Sales Copy

Headlines

The main headline is in the <h1> tag in the Hero section. Keep the <span class="text-cta-red"> wrapper around words you want highlighted in red.

Chapters

Each chapter is a <li> item in the "What You'll Learn" section. Copy a block to add more chapters.

Testimonials

Each testimonial is inside a <blockquote> tag. Edit the quote, name, and title.

Pricing

Find the pricing section and edit the original price (line-through) and sale price. Update the buy button text to match.

7. Countdown Timer

Open assets/js/script.js and find this line near the top:

const COUNTDOWN_DEADLINE = '2026-03-20T23:59:59';

Change the date to your desired deadline. Format: YYYY-MM-DDTHH:MM:SS.

The timer displays in DD:HH:MM:SS format and shows "EXPIRED" when the deadline passes.

8. Buy Button & Payment Link

Find the buy button and replace href="#" with your payment link:

<a href="https://your-payment-provider.com/checkout" id="buy-btn" class="...">
  BUY NOW — Only $27 &rarr;
</a>

Supported platforms: Hotmart, Gumroad, Stripe, PayPal, Kiwify, Eduzz.

9. Changing Colors

colors: {

    muted: '#4A4A4A',      // Secondary text
    light: '#F9F9F9',      // Section backgrounds

  }
}

5. Editando o Texto de Vendas

Títulos

O título principal está na tag <h1> na seção Hero. Mantenha o <span class="text-cta-red"> em volta das palavras que quer destacar em vermelho.

Capítulos

Cada capítulo é um <li> na seção "O Que Você Vai Aprender". Copie um bloco para adicionar mais.

Depoimentos

Cada depoimento está dentro de uma tag <blockquote>. Edite a citação, nome e cargo.

Preço

Encontre a seção de preço e edite o preço original (line-through) e o promocional. Atualize o texto do botão.
