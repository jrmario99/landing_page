1. File Structure

apex/ ├── index.html ← Main page ├── documentation.html ← This file └── assets/ ├── css/ │ └── style.css ← Custom styles (extends Tailwind) ├── js/ │ └── main.js ← All interactivity (Vanilla JS) └── img/ ├── favicon.svg ← Browser tab icon └── (add your images here)

2. Changing Colors (Brand)

Open index.html and find the Tailwind config script near the top:

<script> tailwind.config = { theme: { extend: { colors: { apex: { 500: '#3478ff', // Main brand color 600: '#1b57f5', // Buttons, links 700: '#1443e1', // Hover states } } } } } </script>

Replace the hex values (#3478ff) with your own brand colors. All components using apex-* classes will update automatically.

3. Navbar & Mega Menu

The navbar is transparent on the hero section and becomes solid white on scroll. To modify the mega menu links:

  • Find id="megaMenu" in index.html
  • Each column contains <a class="mega-link"> blocks — edit the text and href
  • The promo banner is the 3rd column — change the text and link

4. Hero Slider

The slider has 3 slides with auto-play (5 second interval). To customize:

  • Find each <div class="hero-slide"> block
  • Edit headings, paragraphs, and button text/links
  • Replace image placeholders with real images
  • To add/remove slides, duplicate or delete a hero-slide block and update the data-index and dots

To change auto-play speed, edit main.js line with setInterval(nextSlide, 5000) — value is in milliseconds.

5. Services Section

Each service card uses Lucide icons. To change an icon:

<!-- Replace the icon name in data-lucide --> <i data-lucide="pen-tool" class="w-6 h-6"></i> <!-- Browse icons at: https://lucide.dev/icons -->

The "lift" hover effect (moves up 5px + colored shadow) is handled by Tailwind classes: hover:-translate-y-1.5 hover:shadow-xl

6. Fun Facts (Counters)

Counters animate when the section enters the viewport using Intersection Observer. To change values:

<span class="counter" data-target="350">0</span> <!-- Change data-target to your desired number -->

7. Portfolio Filter

Portfolio items are filtered by category. To add a new item:

<div class="portfolio-item group" data-category="design"> <!-- Your content here --> </div> <!-- data-category must match one of the filter button data-filter values -->

To add a new category, add a filter button:

<button class="filter-btn" data-filter="branding">Branding</button>

8. CTA Parallax

The parallax effect is a subtle vertical movement of the background. To use a real background image instead of the gradient:

<!-- In index.html, find id="parallaxBg" and add: --> style="background-image: url('assets/img/cta-bg.jpg'); background-size: cover; background-position: center;"

9. Footer & Newsletter

  • Edit the 4 footer columns directly in the HTML
  • Social links: replace href="#" with your actual profile URLs
  • Newsletter form: currently logs to console — connect to your email service (Mailchimp, ConvertKit, etc.) by replacing the submit handler in main.js

10. Replacing Images

All image placeholders include recommended dimensions:

About Image640 x 800pxWebP / JPG
Portfolio Items640 x 480pxWebP / JPG
OG Image1200 x 630pxJPG

11. Deployment

This template uses Tailwind CSS via CDN for quick preview. For production:

  1. Install Tailwind CSS locally: npm install tailwindcss
  2. Create a tailwind.config.js with the color definitions from the CDN script
  3. Build a minified CSS file for optimal performance
  4. Upload all files to your hosting (Netlify, Vercel, shared hosting, etc.)

2. Trocar Cores (Marca)

Abra index.html e encontre o script de configuração do Tailwind no topo:

<script> tailwind.config = { theme: { extend: { colors: { apex: { 500: '#3478ff', // Cor principal 600: '#1b57f5', // Botões, links 700: '#1443e1', // Estados hover } } } } } </script>

Substitua os valores hex (#3478ff) pelas cores da sua marca. Todos os componentes usando classes apex-* serão atualizados automaticamente.

3. Navbar e Mega Menu

A navbar é transparente no hero e fica branca ao rolar. Para modificar os links do mega menu:

  • Encontre id="megaMenu" no index.html
  • Cada coluna contém blocos <a class="mega-link"> — edite o texto e o href
  • O banner promocional é a 3ª coluna — altere o texto e o link

4. Hero Slider

O slider tem 3 slides com autoplay (5 segundos). Para personalizar:

  • Encontre cada bloco <div class="hero-slide">
  • Edite títulos, parágrafos e textos/links dos botões
  • Substitua os placeholders de imagem por imagens reais
  • Para adicionar/remover slides, duplique ou delete um bloco hero-slide e atualize o data-index e os dots

Para alterar a velocidade, edite no main.js a linha setInterval(nextSlide, 5000) — valor em milissegundos.

5. Seção de Serviços

Cada card usa ícones Lucide. Para trocar um ícone:

<!-- Troque o nome do ícone em data-lucide --> <i data-lucide="pen-tool" class="w-6 h-6"></i> <!-- Catálogo de ícones: https://lucide.dev/icons -->

O efeito "lift" no hover (sobe 5px + sombra colorida) é feito pelas classes Tailwind: hover:-translate-y-1.5 hover:shadow-xl

6. Fun Facts (Contadores)

Os contadores animam quando a seção entra na viewport (Intersection Observer). Para trocar valores:

<span class="counter" data-target="350">0</span> <!-- Altere data-target para o número desejado -->

7. Portfólio com Filtro

Os itens do portfólio são filtrados por categoria. Para adicionar um novo item:

<div class="portfolio-item group" data-category="design"> <!-- Seu conteúdo aqui --> </div> <!-- data-category deve corresponder ao data-filter de um botão -->

Para adicionar uma nova categoria, crie um botão de filtro:

<button class="filter-btn" data-filter="branding">Branding</button>

8. CTA Parallax

O efeito parallax é um movimento vertical sutil do fundo. Para usar uma imagem real ao invés do gradiente:

<!-- No index.html, encontre id="parallaxBg" e adicione: --> style="background-image: url('assets/img/cta-bg.jpg'); background-size: cover; background-position: center;"

9. Footer e Newsletter

  • Edite as 4 colunas do footer diretamente no HTML
  • Links sociais: substitua href="#" pelas URLs reais dos seus perfis
  • Newsletter: atualmente faz log no console — conecte ao seu serviço de email (Mailchimp, ConvertKit, etc.) substituindo o handler de submit no main.js

Este template usa Tailwind CSS via CDN para preview rápido. Para produção:

  1. Instale o Tailwind CSS localmente: npm install tailwindcss
  2. Crie um tailwind.config.js com as definições de cores do script CDN
  3. Gere um CSS minificado para melhor performance
  4. Faça upload de todos os arquivos para seu hosting (Netlify, Vercel, hospedagem compartilhada, etc.)
