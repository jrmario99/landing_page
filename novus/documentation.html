Introduction

Thank you for purchasing Novus - Modern SaaS & App Tech Template. This document covers everything you need to get started, customize the template to your brand, and deploy it to production.

Novus is built with modern web technologies including Tailwind CSS v3, Vanilla JavaScript, and Lucide Icons. No build step is required for the preview version (CDN-based), but we recommend a Tailwind build pipeline for production.

Installation

Quick Start (CDN)

The template works out of the box with CDN dependencies. Simply open index.html in your browser.

Production Setup (Recommended)

For production, install Tailwind CSS locally for optimized builds:

npm init -y
npm install -D tailwindcss
npx tailwindcss init

Configure your tailwind.config.js to scan your HTML files, then build:

npx tailwindcss -i ./assets/css/input.css -o ./assets/css/output.css --minify

Update the <link> tag in your HTML to point to output.css and remove the CDN script tag.

Colors & Theming

All colors are defined as CSS custom properties in assets/css/input.css under :root and .dark selectors. To change the brand color, update these variables:

VariableLight ModeDark ModeUsage
--primary#6366f1#818cf8Buttons, links, accents
--accent#06b6d4#22d3eeSecondary accent color
--bg-primary#ffffff#0f172aMain background
--text-primary#0f172a#f1f5f9Headings, body text

Typography

The template uses Plus Jakarta Sans from Google Fonts. To change the font, update the Google Fonts link in the HTML <head> and the font-family in the Tailwind config and CSS.

Components

Key reusable components and their CSS classes:

  • .btn-primary / .btn-secondary - Button styles
  • .glass-card - Glassmorphism card with blur effect
  • .section-badge - Section label badges
  • .section-title - Responsive section headings
  • .pricing-card / .pricing-card.featured - Pricing cards
  • .testimonial-card - Testimonial review cards
  • .tab-btn / .tab-panel - Tabbed interface

Dark Mode

Dark mode is toggled by adding/removing the dark class on the <html> element. The user's preference is saved to localStorage under the key novus-theme. The toggle automatically respects the user's OS preference on first visit.

JavaScript API

All JavaScript is contained in assets/js/main.js using an IIFE pattern (no global scope pollution). Key modules:

  • Theme Toggle - Saves preference to localStorage
  • Navbar - Adds glass effect on scroll
  • Mobile Menu - Slide-in drawer with body scroll lock
  • AOS - Custom IntersectionObserver-based scroll animations
  • Feature Tabs - Click-to-switch content panels
  • Timeline - Progress bar that fills on scroll
  • Counters - Animated number count-up with easing
  • Pricing Toggle - Monthly/Yearly price switch

Deployment

Novus is a static HTML template and can be deployed on any hosting provider:

  • Netlify - Drag and drop the project folder
  • Vercel - Import from Git or upload
  • GitHub Pages - Push to a repository and enable Pages
  • Any web server - Upload files via FTP/SFTP
