1. Overview1. Visao Geral

A minimal, elegant link-in-bio landing page template. Features a profile card, customizable link list, newsletter signup form, and dark/light mode toggle. Built with pure CSS and vanilla JavaScript — zero dependencies.

Um template minimalista e elegante de landing page link-in-bio. Inclui card de perfil, lista de links personalizavel, formulario de newsletter e alternancia de modo escuro/claro. Construido com CSS puro e JavaScript vanilla — zero dependencias.

5. Editing Profile5. Editando o Perfil

In index.html, find the .profile-card section:

  • Avatar — Replace the .avatar-fallback initials or uncomment the <img> tag
  • Name — Edit the <h1> text
  • Tagline — Edit the .profile-tagline paragraph
  • Location — Edit the .profile-location text

No index.html, encontre a secao .profile-card:

  • Avatar — Substitua as iniciais em .avatar-fallback ou descomente a tag <img>
  • Nome — Edite o texto do <h1>
  • Descricao — Edite o paragrafo .profile-tagline
  • Localizacao — Edite o texto em .profile-location

6. Managing Links6. Gerenciando Links

Each link is an <a class="link-item"> element inside .links-section. To add a new link, duplicate an existing one and change the href, icon SVG, and label text.

<a href="https://your-url.com" class="link-item" target="_blank" rel="noopener noreferrer">
  <svg ...>...</svg>
  <span>Link Label</span>
</a>

Cada link e um elemento <a class="link-item"> dentro de .links-section. Para adicionar um novo link, duplique um existente e altere o href, o SVG do icone e o texto do label.

<a href="https://sua-url.com" class="link-item" target="_blank" rel="noopener noreferrer">
  <svg ...>...</svg>
  <span>Texto do Link</span>
</a>

7. Newsletter Form7. Formulario Newsletter

The form uses client-side validation via script.js. To connect it to a real service (Mailchimp, ConvertKit, etc.), update the form's action attribute or add your API integration in the handleSubmit function inside assets/js/script.js.

O formulario usa validacao client-side via script.js. Para conectar a um servico real (Mailchimp, ConvertKit, etc.), atualize o atributo action do formulario ou adicione sua integracao de API na funcao handleSubmit em assets/js/script.js.

8. Dark Mode8. Modo Escuro

Theme preference is saved to localStorage (key: theme) and respects the OS prefers-color-scheme on first visit. The toggle button switches between sun and moon icons.

A preferencia de tema e salva no localStorage (chave: theme) e respeita o prefers-color-scheme do sistema na primeira visita. O botao alterna entre icones de sol e lua.

9. Colors & Typography9. Cores & Tipografia

All colors are defined as CSS custom properties in assets/css/style.css under :root. Edit these values to match your brand:

:root {
  --bg: #faf8f5;
  --text: #2d2d2d;
  --accent: #9b59b6;
  /* ... dark mode overrides in [data-theme="dark"] */
}

Fonts: Playfair Display (headings) and Lora (body text). Change them in the Google Fonts <link> tag and the CSS font-family declarations.

Todas as cores sao definidas como variaveis CSS em assets/css/style.css sob :root. Edite esses valores para combinar com sua marca:

:root {
  --bg: #faf8f5;
  --text: #2d2d2d;
  --accent: #9b59b6;
  /* ... overrides modo escuro em [data-theme="dark"] */
}

Fontes: Playfair Display (titulos) e Lora (corpo). Altere na tag <link> do Google Fonts e nas declaracoes font-family do CSS.
