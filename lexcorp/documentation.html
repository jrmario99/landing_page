2. Branding & Colors

Open assets/css/style.css and find the :root block at the top. Change the Navy and Gold variables:

/* Navy Blue — main dark color */
--navy-900: #0d1a30;

/* Gold — accent highlights */
--gold-500: #c49a38;

The fonts are set to Playfair Display (headings) and Inter (body). Change them in the --font-serif and --font-sans variables, and update the Google Fonts link in index.html.

3. Top Bar & Navbar

  • Phone/email: Edit the links inside class="topbar__left"
  • Business hours: Edit the text in class="topbar__right"
  • Navigation links: Edit href and text in class="header__nav"
  • CTA button: Change the text in class="header__cta"

The navbar is sticky by default and adds a shadow on scroll via JavaScript.

4. Hero Section & Quick Form

The hero uses a dark overlay pattern. To add a real background image:

/* In style.css, add to .hero: */
background-image: url('assets/img/hero-bg.jpg');
background-size: cover;
background-position: center;

The Quick Case Analysis form floats on the right. It currently logs to console — connect it to your backend or a service like Formspree/Netlify Forms by modifying the submit handler in main.js.

5. Trust Badges

Replace the placeholder boxes with actual logos of bar associations, awards, or certifications. Use grayscale images for a professional look. Recommended size: 56×56px or SVG.

6. Practice Areas (Tabs)

Vertical tabs on desktop, horizontal scrollable tabs on mobile. To add a new area:

  1. Add a new <button class="practice__tab" data-tab="newtab">
  2. Add a corresponding <div class="practice__panel" id="panel-newtab">
  3. The JavaScript handles switching automatically — no code changes needed

7. Attorney Team

  • Replace the image placeholder with real photos (recommended: 400×500px)
  • Edit name, role, and OAB number in attorney-card__info
  • Update the LinkedIn href="#" with actual profile URLs
  • On hover, a "Schedule Consultation" button appears over the photo

8. Case Results

Each card shows a monetary value or outcome, the practice area, a title, and a description. Edit them directly in the HTML. To add more cards, duplicate a result-card block.

Important: Ensure all case descriptions comply with your local bar association's advertising rules (OAB in Brazil).

9. FAQ Accordion

The accordion uses animated +/- icons with smooth height transitions. To add a new question:

<div class="accordion">
  <button class="accordion__trigger" aria-expanded="false">
    <span>Your question here?</span>
    <div class="accordion__icon"><span></span><span></span></div>
  </button>
  <div class="accordion__content">
    <p>Your answer here.</p>
  </div>
</div>

10. Contact & Map

Changing the map:

  1. Go to Google Maps
  2. Search your address → Click Share → Embed a map
  3. Copy the src="..." URL
  4. Find <!-- EDITABLE: Google Maps Embed --> in index.html
  5. Replace the src value in the <iframe>

Contact form:

Currently logs to console. Connect to your backend or email service by editing the form handler in main.js.

├── index.html              ← Página principal

        ├── favicon.svg     ← Ícone da aba

2. Marca e Cores

Abra assets/css/style.css e encontre o bloco :root no topo. Altere as variáveis Navy e Gold:

/* Azul Marinho — cor escura principal */
--navy-900: #0d1a30;

/* Dourado — destaques de acento */
--gold-500: #c49a38;

As fontes são Playfair Display (títulos) e Inter (corpo). Altere em --font-serif e --font-sans, e atualize o link do Google Fonts no index.html.

3. Barra Superior e Navbar

  • Telefone/email: Edite os links dentro de class="topbar__left"
  • Horário de funcionamento: Edite o texto em class="topbar__right"
  • Links de navegação: Edite href e texto em class="header__nav"
  • Botão CTA: Altere o texto em class="header__cta"

A navbar é sticky por padrão e ganha sombra ao rolar via JavaScript.

4. Hero e Formulário Rápido

O hero usa um overlay escuro com padrão de grid. Para adicionar uma imagem de fundo real:

/* No style.css, adicione em .hero: */
background-image: url('assets/img/hero-bg.jpg');
background-size: cover;
background-position: center;

O formulário de Análise de Caso Rápida flutua à direita. Atualmente faz log no console — conecte ao seu backend ou serviço como Formspree editando o handler de submit no main.js.

5. Selos de Confiança

Substitua os placeholders por logos reais de OAB, associações ou prêmios. Use imagens em tons de cinza para um visual profissional. Tamanho recomendado: 56×56px ou SVG.

6. Áreas de Atuação (Abas)

Abas verticais no desktop, horizontais com scroll no mobile. Para adicionar uma nova área:

  1. Adicione um novo <button class="practice__tab" data-tab="novaaba">
  2. Adicione o painel correspondente <div class="practice__panel" id="panel-novaaba">
  3. O JavaScript gerencia a troca automaticamente — sem necessidade de alterar código

7. Equipe de Advogados

  • Substitua os placeholders por fotos reais (recomendado: 400×500px)
  • Edite nome, cargo e número OAB em attorney-card__info
  • Atualize o href="#" do LinkedIn com as URLs reais dos perfis
  • No hover, um botão "Agendar Consulta" aparece sobre a foto

8. Resultados de Casos

Cada card mostra um valor ou resultado, a área de atuação, um título e uma descrição. Edite diretamente no HTML. Para adicionar mais cards, duplique um bloco result-card.

Importante: Certifique-se de que todas as descrições de casos estejam em conformidade com as normas de publicidade da OAB.

9. FAQ Accordion

O accordion usa ícones +/- animados com transição suave de altura. Para adicionar uma nova pergunta:

<div class="accordion">
  <button class="accordion__trigger" aria-expanded="false">
    <span>Sua pergunta aqui?</span>
    <div class="accordion__icon"><span></span><span></span></div>
  </button>
  <div class="accordion__content">
    <p>Sua resposta aqui.</p>
  </div>
</div>

10. Contato e Mapa

Trocar o mapa:

  1. Acesse Google Maps
  2. Pesquise seu endereço → Clique em Compartilhar → Incorporar um mapa
  3. Copie a URL src="..."
  4. Encontre <!-- EDITABLE: Google Maps Embed --> no index.html
  5. Substitua o valor src no <iframe>

Formulário de contato:

Atualmente faz log no console. Conecte ao seu backend ou serviço de email editando o handler do formulário no main.js.
