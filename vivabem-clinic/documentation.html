1. Overview

VivaBem Clinic is a single-page landing page template designed for health clinics, medical offices, gyms, and local businesses. It features a clean, modern design with sections for services, testimonials, business hours, location map, and a contact form.

The template is fully responsive, uses vanilla CSS and JavaScript (zero dependencies), and is built with accessibility and SEO best practices in mind.

4. Page Sections

  • Hero — Full-width banner with headline, subtext, dual CTA buttons, and placeholder image

5. Changing Colors

Open assets/css/style.css and find the :root block near the top:

:root {
    /* ── EDITABLE: Brand Colors ── */
    --primary: #2563eb;        /* Main blue */
    --primary-dark: #1d4ed8;   /* Hover blue */
    --primary-light: #dbeafe;  /* Light blue backgrounds */
    --accent: #10b981;         /* Green accent */
    --accent-dark: #059669;    /* Hover green */
}

Change these values to match your brand. All buttons, links, and highlights will update automatically.

6. Google Maps Location

  1. Go to Google Maps
  2. Search for your business address
  3. Click Share → Embed a map → Copy HTML
  4. In index.html, find the comment <!-- EDITABLE: Google Maps Embed -->
  5. Replace the src="..." inside the <iframe> with the new URL
Also update the <script type="application/ld+json"> block with your actual latitude, longitude, and address for SEO.

7. Business Hours

In the HTML

Find the comment <!-- EDITABLE: Business Hours --> and edit each <div class="hours__row"> block. Add the class hours__row--closed for closed days.

In the JavaScript

Open assets/js/main.js and find the schedule object (around line 65):

const schedule = {
    0: null,                              // Sunday — Closed
    1: { open: "07:00", close: "20:00" }, // Monday
    2: { open: "07:00", close: "20:00" }, // Tuesday
    ...
    6: { open: "08:00", close: "14:00" }, // Saturday
};

Use null for closed days and 24-hour format "HH:MM".

In Schema.org

Update the openingHoursSpecification array in the JSON-LD block to match your real hours.

8. WhatsApp Number

  1. Open index.html
  2. Search for 5511999999999 (appears 2 times)
  3. Replace with your number in the format: country code + area code + number (no spaces or dashes)
Example for Brazil: 5521987654321

9. Social Media Links

Find the comment <!-- EDITABLE: Social Media Links --> in index.html and replace each href="#" with your actual profile URLs (Instagram, Facebook, etc.).

10. SEO & Schema.org

  • Meta tags — Title, description, Open Graph and Twitter Card tags are pre-configured in the <head>
  • Semantic HTML — Uses <header>, <main>, <section>, <footer>, and proper heading hierarchy
  • Schema.org — LocalBusiness JSON-LD structured data with name, address, phone, hours, and geo coordinates
  • Social sharing image — Add an og-image.jpg (1200×630px) to assets/img/
Remember to replace placeholder URLs (clinicavivabem.com.br) with your actual domain before publishing.

11. Performance

  • Zero external JS dependencies — only vanilla JavaScript
  • CSS custom properties for efficient theming
  • Google Maps iframe uses loading="lazy"
  • Font preconnect hints for faster Google Fonts loading
  • Intersection Observer for scroll-triggered animations (no scroll event polling)
  • Phone number input mask using lightweight vanilla JS

1. Visão Geral

VivaBem Clinic é um template de landing page single-page projetado para clínicas de saúde, consultórios médicos, academias e negócios locais. Possui um design limpo e moderno com seções de serviços, depoimentos, horários de funcionamento, mapa de localização e formulário de contato.

O template é totalmente responsivo, usa CSS e JavaScript puros (zero dependências), e foi construído seguindo boas práticas de acessibilidade e SEO.

5. Trocar as Cores

Abra assets/css/style.css e encontre o bloco :root no topo:

:root {
    /* ── EDITABLE: Brand Colors ── */
    --primary: #2563eb;        /* Azul principal */
    --primary-dark: #1d4ed8;   /* Azul hover */
    --primary-light: #dbeafe;  /* Fundos azul claro */
    --accent: #10b981;         /* Verde destaque */
    --accent-dark: #059669;    /* Verde hover */
}

Altere esses valores para as cores da sua marca. Todos os botões, links e destaques serão atualizados automaticamente.

6. Localização no Google Maps

  1. Acesse Google Maps
  2. Pesquise o endereço do seu negócio
  3. Clique em Compartilhar → Incorporar um mapa → Copiar HTML
  4. No index.html, encontre o comentário <!-- EDITABLE: Google Maps Embed -->
  5. Substitua o src="..." dentro do <iframe> pela nova URL
Atualize também o bloco <script type="application/ld+json"> com sua latitude, longitude e endereço reais para SEO.

7. Horários de Funcionamento

No HTML

Encontre o comentário <!-- EDITABLE: Business Hours --> e edite cada bloco <div class="hours__row">. Adicione a classe hours__row--closed para dias fechados.

No JavaScript

Abra assets/js/main.js e encontre o objeto schedule (por volta da linha 65):

const schedule = {
    0: null,                              // Domingo — Fechado
    1: { open: "07:00", close: "20:00" }, // Segunda
    2: { open: "07:00", close: "20:00" }, // Terça
    ...
    6: { open: "08:00", close: "14:00" }, // Sábado
};

Use null para dias fechados e formato 24 horas "HH:MM".

No Schema.org

Atualize o array openingHoursSpecification no bloco JSON-LD com seus horários reais.

8. Número do WhatsApp

  3. Substitua pelo seu número no formato: código do país + DDD + número (sem espaços ou traços)
Exemplo: 5521987654321
