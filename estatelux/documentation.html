1. Overview

EstateLux is a premium real estate template designed with a "Modern Luxury" aesthetic — black marble vibes, gold accents, and generous whitespace. Built for high-end property listings, it features a robust dark mode, glassmorphism search bar, interactive map pins, masonry gallery with lightbox, and a floating WhatsApp conversion button.

3. Pages

index.html — Homepage

  • Hero: Full-screen background image with dark gradient overlay. Glassmorphism search bar with Buy/Rent tabs, location input, type dropdown, and dual-range price slider. Animated stats counters.
  • Featured Properties: 6 property cards in a responsive grid with image hover zoom, favorite heart button, badges (Exclusive/New), price overlay, and specs (bed/bath/sqft).
  • Map Section: Visual map placeholder with interactive pin markers showing property prices on hover.
  • About / Trust: Company info section with features grid, CTA button, and "As Seen In" logos (Forbes, ArchDaily, VEJA, GQ, Exame).
  • Newsletter CTA: Dark section with email subscription form.

property-detail.html — Property Detail

  • Masonry Gallery: 5-image grid (main spanning 2x2) that opens a full-screen lightbox with prev/next, keyboard navigation, and touch swipe.
  • Property Info: Title, badges, location, price, IPTU/condo fees, quick specs (4 icons), description, amenities grid (6 items with icons), video tour placeholder, and map.
  • Sticky Agent Sidebar: Agent card with photo, name, CRECI, WhatsApp button, call button, and schedule visit button. Stays fixed while scrolling.

4. Dark Mode Strategy

EstateLux implements a dual-personality design:

Light Mode Dark Mode
BackgroundOff-white (#fafafa)Rich black (#0c0a09)
TextDark grey (#292524)Light (#e7e5e4)
Primary CTAEmerald Green (#047857)Gold/Amber (#d4a012)
CardsWhite (#ffffff)Dark (#1c1917)
AccentsEmerald green (#047857)Gold (#d4a012)
 
// Stored as:
localStorage.setItem('estatelux-theme', 'dark' | 'light');

// Respects OS preference on first visit:
window.matchMedia('(prefers-color-scheme: dark)')

5. Customization

Colors

// In tailwind.config (inside each HTML file):
colors: {
    brand: {
        emerald: '#047857',      // Light mode primary
        gold: '#d4a012',         // Dark mode primary
    }
}

// Also update style.css where hex values appear

WhatsApp Number

Replace 5511999999999 in all wa.me links with your actual WhatsApp number.

Images

  • Hero: 1920×1080px
  • Property cards: 600×450px (4:3)
  • Gallery main: 800×600px
  • Gallery thumbs: 400×300px
  • Agent photo: 120×120px

Map Integration

The map section uses a placeholder image. To add a real map, replace the image with a Google Maps, Mapbox, or Leaflet embed.

6. Key Features

  • Floating WhatsApp Button: Fixed bottom-right, pulsing green animation. Pre-filled message.
  • Glassmorphism Search: Blur background, Buy/Rent tabs, price range slider with real-time BRL formatting.
  • Property Cards: Image zoom on hover, favorite toggle, badges, responsive specs.
  • Lightbox Gallery: Full-screen with keyboard nav, touch swipe, fade transitions.
  • Sticky Agent Card: Remains visible while scrolling. WhatsApp, phone, schedule.
  • Animated Counters: Numbers count up with easeOutCubic when visible.

7. SEO & Semantics

  • <article> — Property cards and detail content
  • <aside> — Agent sidebar
  • <nav aria-label> — All navigation with ARIA labels
  • <header> / <main> / <footer> — Page landmarks
  • Open Graph + Twitter Card meta tags included
  • All images have descriptive alt text
  • All buttons have aria-label attributes

8. Performance

  • Zero dependencies — Tailwind CDN + vanilla JS only
  • Lazy loading — All non-hero images lazy loaded
  • Passive scroll listeners
  • Font preconnect
  • Staggered scroll reveal
  • Print styles included
